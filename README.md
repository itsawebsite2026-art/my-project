# 🚀 ORI - Job Tracking System
ระบบบริหารจัดการงานผลิต (Job Tracking) ที่พัฒนาด้วย Google Apps Script, React และ Tailwind CSS

![App Mockup Preview](https://github.com/itsawebsite2026-art/my-project/raw/main/preview.png)

## ✨ คุณสมบัติเด่น (Features)
- **Modern App Interface**: ดีไซน์แบบ App Window Mockup ทันสมัย พร้อม MacOS-style controls
- **Real-time Synchronization**: เชื่อมต่อข้อมูลโดยตรงกับ Google Sheets แบบเรียลไทม์
- **Role Separation**: ระบบแยกหน้าที่ชัดเจนระหว่าง "ผู้สร้างใบงาน" และ "ผู้แก้ไขล่าสุด"
- **Urgency Tracking**: ระบบติดตามสถานะงาน (รอดำเนินการ, กำลังทำ, เสร็จสิ้น) พร้อมการเน้นสีตามความสำคัญ
- **Material Calculator**: เครื่องมือคำนวณราคาวัสดุและพิมพ์ในตัว
- **Background Uploads**: ระบบอัปโหลดรูปภาพในพื้นหลัง ไม่ขัดจังหวะการทำงาน

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- **Frontend**: React.js, Tailwind CSS, Lucide Icons
- **Backend**: Google Apps Script (GAS)
- **Database**: Google Sheets
- **Deployment**: Google Apps Script Web App

## 📦 การติดตั้งและใช้งาน (Installation)
1. คัดลอกโค้ดไปวางใน Google Apps Script Project
2. ตั้งชื่อไฟล์ให้ตรงกับโครงสร้าง (Index.html, Code.js, ฯลฯ)
3. ปรับเปลี่ยน `SHEET_ID` ใน `Code.js` ให้เป็น ID ของสเปรดชีตของคุณ
4. สั่ง Deploy เป็น Web App
