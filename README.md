## 💎 ทำไมต้อง ORI?
ORI ไม่ใช่แค่ระบบบันทึกงานทั่วไป แต่ถูกออกแบบมาให้เน้น **User Experience (UX)** ที่เหมือนกับแอปพลิเคชันระดับไฮเอนด์ (Premium App Mockup) เพื่อให้ทีมงานของคุณทำงานได้อย่างลื่นไหลและผิดพลาดน้อยที่สุด

### 🌟 ฟีเจอร์ที่เหนือกว่า (Key Features)
- **🎨 Premium UI Design**: เลย์เอาต์แบบ MacOS Window Mockup พร้อมปุ่ม Traffic Light ที่ให้ความรู้สึกเหมือนแอปพลิเคชันจริง
- **🔐 Advanced Role Management**: ระบบแยกหน้าที่ชัดเจนระหว่าง **Creator** (ผู้สร้างงาน) และ **Updater** (คนอัปเดตงาน) ป้องกันการเขียนทับข้อมูลประวัติ
- **⚡ Real-time Google Sheets Sync**: ข้อมูลเก็บไว้ใน Google Sheets ของคุณเอง ปลอดภัยและเข้าถึงได้จากทุกที่
- **📊 Urgency Indicators**: ระบบเน้นสีงานตามความสำคัญและสถานะ (รอดำเนินการ, กำลังทำ, เสร็จสิ้น) ช่วยให้ไม่พลาดทุกกำหนดส่ง
- **📸 Intelligent Background Upload**: อัปโหลดรูปภาพงานในพื้นหลัง ไม่ต้องรอให้เสร็จก่อนถึงจะทำงานต่อได้

## 🛠️ เทคโนโลยี (Tech Stack)
- **Core**: React.js 18 + Google Apps Script
- **Styling**: Tailwind CSS (Premium Dark/Light Mode support)
- **Icons**: Lucide React
- **Backend**: JavaScript (Google Apps Script API)

---

## 📦 การติดตั้งและใช้งาน (Installation)
1. **เตรียมไฟล์**: คัดลอกไฟล์ทั้งหมด (index.html, Code.js, ฯลฯ) ไปไว้ใน Google Apps Script Project
2. **ตั้งค่าไอดี**: แก้ไข `SHEET_ID` ในไฟล์ `Code.js` ให้เป็นไอดีของ Google Sheets ของคุณ
3. **Deploy**: สั่ง `Deploy > New Deployment` เป็นประเภท **Web App**
4. **เปิดใช้งาน**: นำ URL ที่ได้ไปแชร์ให้ทีมงานใช้งานได้ทันที
